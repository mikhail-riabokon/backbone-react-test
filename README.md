## Backbone + React test integration based on [Pipedrive API](https://developers.pipedrive.com/docs/api/v1/)

### Before you start
* Install Node (```6.*.*```)
* Install npm (```3.*.*```)

### How to start
* Clone project
* Go to the project
* Change Pipedrive API key (REACT_APP_API_KEY) in .env file to yours
* ```npm install```
* ```npm start```
